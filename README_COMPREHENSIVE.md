```bash

docker compose up -d

# 4. Watch the logs
docker compose logs -f civicverse-frontend civicverse-backend multiplayer-server

# 5. Open your browser
# Frontend: http://localhost:3000
# API: http://localhost:3001/api/stats
# Multiplayer: ws://localhost:8080/ws

# 6. Test multiplayer
# Open http://localhost:3000 in 2+ browser tabs
# WASD to move, SPACE to attack
```

#### **Option B: Local Development (Node.js)**

```bash
# 1. Clone and navigate
git clone https://github.com/Civicverse/Foyer.git
cd Civicverse

# 2. Install backend dependencies
cd backend
npm install
node index.js  # Starts on port 3001

# 3. In a new terminal, start multiplayer server
cd backend
node multiplayer-server.js  # Starts on port 8080

# 4. In a new terminal, start frontend
cd frontend/demo
npm install
npm run dev  # Starts on http://localhost:5173

# 5. Open http://localhost:5173 in your browser
```
