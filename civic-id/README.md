# Civic ID (Prototype)

Civic ID is a local-first, non-custodial identity and avatar prototype.

## Features
- Deterministic Civic ID generation
- Avatar creation and progression
- Wallet balance tracking
- Trust / vouch system
- Local JSON persistence

## Run

```bash
npm install
npm run start

Save and exit.
